"عباد الرحمن" – رفيقك اليومي للعبادة والطاعة

الله يلعن الجن (عموما) و اولاهم الشيطان بداخلى الذى يقول انه خان ابليس  و اهلة وذريته و الانس من رضى بالظلم فى الدنيا و الاخرى مسلم او كافر 

صُمم تطبيق "عباد الرحمن" بعناية ليجمع بين جمال التصميم وسهولة الاستخدام، موفراً لكل مسلم الأدوات الأساسية التي يحتاجها في يومه. واجهة عصرية وهادئة تأخذك في رحلة إيمانية خالية من المشتتات.

✨ ميزة جديدة: تفاصيل شاملة للأذكار قمنا بإثراء مكتبة الأذكار! الآن، كل ذكر ودعاء يحتوي على تفاصيل دقيقة تشمل "المناسبة" (متى يُقال) و "الفضل" (الثواب الوارد في السنة)، يمكن الوصول إليها بلمسة واحدة على أيقونة المعلومات. تذوق حلاوة الذكر وافهم معانيه.

أهم المميزات:

📍 مواقيت الصلاة بدقة عالية

حسابات دقيقة بناءً على موقعك الجغرافي الحالي.
دعم لمختلف طرق الحساب والمذاهب (الشافعي/الحنفي).
يعمل بدون إنترنت: جميع المميزات تعمل بكفاءة تامة دون الحاجة لاتصال.
🕌 الأذان والتنبيهات الذكية

أذان كامل بصوت عذب: استمع لنداء الصلاة (مكة المكرمة، المدينة المنورة، مصر).
تنبيهات قبل الصلاة: إشعارات ذكية لتذكيرك قبل دخول الوقت للاستعداد الروحي.
موثوقية في الخلفية: التنبيهات تعمل بدقة حتى لو كان التطبيق مغلقاً.
📖 القرآن الكريم

نص قرآني واضح (الرسم العثماني) ومريح للعين.
تلاوات صوتية: استمع لأشهر القراء بجودة عالية.
التفسير الميسر: افهم معاني الآيات بلمسة واحدة.
📿 الأذكار والأدعية الشاملة

أذكار الصباح والمساء: مكتبة كاملة وموثقة مع عداد إلكتروني.
أدعية لكل حال: أدعية النوم، الاستيقاظ، الحماية، والهم والحزن.
المناسبة والفضل: [جديد] تعرف على فضل كل ذكر ومناسبته.
🧭 القبلة والمساجد القريبة

اتجاه القبلة: بوصلة دقيقة وسلسة لتحديد اتجاه الكعبة المشرفة من أي مكان.
البحث عن المساجد: اعثر على أقرب المساجد إليك فوراً بضغطة زر.
📅 أدوات للمؤمن

التقويم الهجري: متابعة التواريخ والمناسبات الإسلامية.
قضاء الفوائت: أداة مخصصة لحساب ومتابعة الصلوات الفائتة لقضائها.
السنن والنوافل: دليل ومتابعة لسنن الصلوات الرواتب والنوافل.
🎨 تجربة مستخدم فريدة

تصميم فاخر: ثيم "الزمرد العميق" المريح للعين مع ألوان ذهبية هادئة.
الوضع الليلي: دعم كامل للوضع الداكن (Dark Mode).
خصوصية تامة: لا إعلانات مزعجة، لا تتبع للبيانات. عبادتك شيء خاص بك.
ويديجت (Widgets): أدوات للشاشة الرئيسية لعرض مواقيت الصلاة بأناقة.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
```sh
git clone <YOUR_GIT_URL>
```

2. Install dependencies
```sh
npm install
```

3. Start the development server
```sh
npm run dev
```

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
